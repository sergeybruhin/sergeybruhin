## What I Build

- Business websites, ecommerce projects, and admin panels.
- Private CRMs, task trackers, uptime checkers, and internal dashboards.
- Booking, catalog, content, education, and workflow tools.
- Laravel/Vue applications for teams that need something custom, stable, and maintainable.

I like mid-budget projects where the business need is clear, the software has to work for years, and steady support matters as much as the first launch.

## Tech Stack

**Backend:** PHP, Laravel, Eloquent, Nova, PHPUnit

**Data & Ops:** SQLite, MySQL, queues, deployment scripts, Linux servers  
**Daily Tools:** Git, Composer, npm, macOS, CLI workflows
